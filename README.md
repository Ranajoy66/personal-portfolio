# 🌐 Personal Portfolio Website

A modern and responsive **developer portfolio website** built to showcase my projects, skills, and experience as a **Web & Software Developer**.

This portfolio highlights my work, technical skills, and provides an easy way for recruiters or collaborators to connect with me.

## ✨ Features

* Responsive design for **desktop, tablet, and mobile devices**
* **Animated hero section** with typing effect
* **Interactive project showcase**
* **Smooth scrolling navigation**
* **Scroll progress indicator**
* **3D hover effects on project cards**
* **Animated section reveal on scroll**
* **Dark-themed modern UI**
* **Contact section for easy communication**

## ⚙️ Setup & Installation

To run this project locally:

1. Clone the repository

```
git clone https://github.com/Ranajoy66/personal-portfolio.git
```

2. Navigate to the project folder

```
cd personal-portfolio
```

3. Open `index.html` in your browser.
